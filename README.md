# CyberSecurityChatBot 

![.NET Build](https://github.com/fr3ddie-rsa/CyberSecurityChatBot/actions/workflows/dotnet.yml/badge.svg)

## Overview
CyberSecurityChatBot is a console application that teaches users about cybersecurity concepts in an interactive way.  
It includes:
- WAV greeting sound
- ASCII art welcome screen
- Colored console responses
- Personalized chatbot interactions

## Getting Started

### Prerequisites
- Windows 10/11
- Visual Studio 2022 (or later)
- .NET 6 SDK (or .NET 8 if targeted)
